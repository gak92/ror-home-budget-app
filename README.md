## Getting Started

### Setup

To setup the project locally, clone the repository using:

```
$ git clone git@github.com:gak92/ror-home-budget-app.git
```

### Install dependencies
run:
```
$ bundle install
```

### Setup the database
run:
```
$ rails db:create
$ rails db:migrate
```

### Run the app
run:
```
$ rails s
```

### Run the tests
run:
```
$ rspec
```
